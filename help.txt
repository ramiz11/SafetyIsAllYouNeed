"no database context specified" error, do the following
1) In command line:
1.1) export PROJ_LIB = path_to_venv/share/proj
1.2) echo 'export PROJ_LIB = path_to_venv/share/proj' >> ~/.bashrc
1.3) projinfo EPSG:4326
** path_to_venv example: /Users/name/miniforge3/envs/poi_env
2) in project code add the following:
2.1) import pyproj
2.2) pyproj.datadir.set_data_dir('path_to_venv/share/proj')
2.3) from pyproj import CRS
